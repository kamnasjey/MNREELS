# MNREELS - UI Page бүтэц & Wireframe

## Bottom Navigation (4 tab)

```
┌────────────┬────────────┬────────────┬────────────┐
│   🏠 Нүүр  │  🔍 Хайх   │  🎬 Тасалбар│  👤 Профайл│
└────────────┴────────────┴────────────┴────────────┘
```

- **Нүүр** — Swipe feed (TikTok маягийн)
- **Хайх** — Хайлт, ангилал, trending
- **Тасалбар** — Тасалбар худалдаж авах, үлдэгдэл харах
- **Профайл** — Миний хуудас, тохиргоо

> Бүтээгчид нэмэлт **"+" Upload** товч гарна (Профайл tab-ын дотор)

│ MNREELS          🔔 [12]        │  ← Header: лого + мэдэгдэл
├─────────────────────────────────┤

│                                 │

├─────────────────────────────────┤
│ 🖼 @username                    │  ← Бүтээгчийн нэр (дарвал профайл)

│             │                   │

### Видео тоглуулагчийн дээрх давхарга (overlay)
- Бүтээгчийн avatar + нэр (дарвал профайл руу)
- Цувралын нэр + ангийн дугаар
- Видеоны тайлбар (1-2 мөр, дарвал бүтэн текст)
- Баруун талд: Like, Comment, Share, Follow товчууд
- Доод талд: Progress bar (видеоны явц)
