# MNREELS - Бүтээгчийн орлого & Payout систем

## Орлогын хуваарилалт
- **85%** бүтээгчид
- **15%** платформд (MNREELS)

## Бүтээгчийн орлогын тооцоо (1 анги = 2 тасалбар = 100₮)

| Metric | Дүн |
|--------|-----|
| Нэг үзэлтийн орлого | 100₮ |
| Бүтээгч авна (85%) | 85₮ |
| Платформ авна (15%) | 15₮ |
| 1,000 үзэлт | 85,000₮ |
| 10,000 үзэлт | 850,000₮ |
| 50,000 үзэлт | 4,250,000₮ |

## Цувралын орлогын жишээ (30 анги)

| Анги бүрийн дундаж үзэлт | Нийт орлого (85%) | Дундаж цалинтай харьцуулалт |
|---------------------------|-------------------|----------------------------|
| 1,000 | 2,550,000₮ | ~1.8x |
| 5,000 | 12,750,000₮ | ~9x |
| 10,000 | 25,500,000₮ | ~18x |

## Мөнгө татах урсгал
```
Бүтээгчийн цугларсан тасалбар
    │
    ▼
Платформтой exchange хүсэлт илгээх
    │
    ▼
Admin баталгаажуулалт
    │
    ▼
Банкны данс руу шилжүүлэг (₮-ээр)
```

## Дүрэм
- **7 хоног тутам** мөнгө татах боломжтой
- Банкны данс руу шилжүүлнэ
- Бүтээгч өөрийн тасалбарыг платформтой exchange хийнэ
- Admin баталгаажуулсны дараа шилжүүлэг хийгдэнэ
- Minimum withdrawal: 50,000₮ (1,000 тасалбар)

## Бүтээгчийн dashboard
- Нийт орлого (тасалбар + ₮)
- Татаж авсан орлого
- Кино/анги тус бүрийн орлого
- Үзэлтийн статистик
- Мөнгө татах хүсэлтийн түүх
