# MNREELS - Бүтээгчийн орлого & Payout систем

## Орлогын хуваарилалт
- **85%** бүтээгчид
- **15%** платформд (MNREELS)

## Мөнгө татах урсгал
```
Бүтээгчийн цугларсан coin
    │
    ▼
Платформтой exchange хүсэлт илгээх
    │
    ▼
Admin баталгаажуулалт
    │
    ▼
Банкны данс руу шилжүүлэг
```

## Дүрэм
- **7 хоног тутам** мөнгө татах боломжтой
- Банкны данс руу шилжүүлнэ
- Бүтээгч өөрийн coin-ыг платформтой exchange хийнэ
- Admin баталгаажуулсны дараа шилжүүлэг хийгдэнэ
- Minimum withdrawal amount тогтооно (судалгаагаар)

## Бүтээгчийн dashboard
- Нийт орлого (coin)
- Татаж авсан орлого
- Кино тус бүрийн орлого
- Үзэлтийн статистик
- Мөнгө татах хүсэлтийн түүх
