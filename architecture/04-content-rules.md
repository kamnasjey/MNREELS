# MNREELS - Контентын дүрэм & Модераци

## Видеоны шаардлага
- **Урт:** 1 - 15 минут
- **Формат:** Цуврал (series) эсвэл бие даасан кино
- Хэн ч бүртгүүлээд content upload хийж болно

## Контент бүтэц
- Дийлэнх контент **цуврал (series)** хэлбэрээр
- Цуврал дотор episode/анги байна
- Бүтээгч эхний X ангийг үнэгүй болгож болно

## Модераци систем
```
Бүтээгч upload хийнэ
    │
    ▼
2 цагийн шалгалтын цонх эхэлнэ
    │
    ├── Admin шалгаад зөвшөөрвөл ──► Шууд нийтлэх
    ├── Admin татгалзвал ──► Нийтлэхгүй (шалтгаан мэдэгдэнэ)
    └── Admin 2 цагийн дотор харахгүй бол ──► Автомат нийтлэх
```

## Насны ангилал
| Зэрэглэл | Тайлбар |
|-----------|---------|
| Бүгд      | Бүх насныханд зориулсан |
| 13+       | 13-аас дээш насныханд |
| 16+       | 16-аас дээш насныханд |
| 18+       | Насанд хүрэгчдэд зориулсан |

## Хүүхдийн контент
- Насанд хүрээгүй хүүхдийн сэдэвт контент тусад нь байрлана
- Тусгай категори/section-тэй байж болно
